Bookmark-App

Live Github

An app to save your bookmarks. Create a new bookmark by pressing the new button. Enter in a title, a rating of 1-5 stars, a description and the bookmark's url. After creating a bookmark, you'll be taken to the home page where you can filter the bookmarks based on the rating the created earlier. Tap on the bookmark's name and you'll be able see a detailed view of the bookmark and to the right, you can delete the bookmark once it has outlasted its usability. Don't want to overburden your web browser with bookmarks? You should use this website.

Google Bookmark search

Live Github

It's an app that searches through Google's Books API and displays a number of books matching your criteria. Here's what you can search for: the Title of the book, the print-type, the book-type (full, partial, free, paid). It's pretty cool. It's made using React. The results show a title, a thumbnail, a price if it's for sale, and a description.
